Build a blog with HEXO and GitHub Page

20-11-2017
blog, git, github page, hexo

#Why am I doing this?

Recent years, wordpress has been changed so much that it does not look like a blogging CMS anymore. Hence, I was looking for alternative that can replace wordpress in blogging feature. After two days googling, I decided to use HEXO + GitHub Page to build my personal blog.

#Why HEXO
I have compared HEXO, Hugo, Jekyll and some other static site generators. Here are some reasons why I choose HEXO.

[x] Easy to setup
[x] Easy to deploy
[x] Quite fast (Not as fast as Hugo but fast enough for me)
[x] Free hosting on GitHub Page / Netlify
